# Mindustry 游戏内容速查表

> **数据来源**：Mindustry v157.4 源码 + `logicids.dat` + `bundle_zh_CN.properties`

## 符号说明

| 符号 | 含义 |
|------|------|
| `*` | **无内容** — 没有对应内容时用此替代 |
| `/` | **隐藏内容** — 在核心数据库中标记为隐藏，通常不在正常游戏中出现 |

## 排序规则

| 内容类型 | 排序依据 | 说明 |
|----------|----------|------|
| 块 / 地板 / 矿 | **内部名称**（字母序） | 这三类属于环境层，无 Logic ID，无法通过 `lookup` 查找 |
| 建筑 / 物品 / 流体 / 单位 / 队伍 | **Logic ID**（数字序） | 使用处理器 `lookup` 指令所需的索引值 |

> **注意**：少数简体中文名称显示为英文，是因为 `bundle_zh_CN.properties` 中未提供对应翻译。
