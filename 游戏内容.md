# 游戏内容参考

本文档列出 Mindustry 中所有可通过逻辑系统使用/查询的游戏内容，按逻辑ID排序。
隐藏内容（在核心数据库中不显示）以 `/` 前缀标记，无简体中文名称以 `*` 代替。

尺寸（Size）为建筑的占地大小（NxN格），单位的尺寸为碰撞半径换算的格子数。
范围（Range）为 Sensor @range 返回的值（格子单位），即建筑/单位的基础作用范围。
带 `+` 的范围表示使用相织布/相位织物强化后的最大范围。
带 `*` 前缀的范围表示该建筑不实现范围接口，Sensor @range 返回 0，但该值为其实际作用范围。
单位的物品传输范围：逻辑指令 itemTake/itemDrop 的有效距离为 5.6 格 + 目标建筑尺寸×0.5 格。
